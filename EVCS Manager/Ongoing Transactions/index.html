    Ongoing Transactions

    [GET] transactions/ongoing

    /api/transactions/ongoing

    OngoingTransactionsRequestDTO

    Attributes:

    Name Type Description
    site_id str

    site id of the ongoing transactions

    OngoingTransactionsResponseDTO

    Attributes:

    Name Type Description
    transactions list

    list of ongoing transactions of type TransactionDTO

    TransactionDTO

    Attributes:

    Name Type Description
    reference_num str

    transaction id
    start_date str

    start date of the transaction in the format of 'YYYY-MM-DD HH:MM:SS'
    charger_name str

    charger used for the transaction
    location str

    location of the transaction
    demand float

    kWh consumed during charging session
    driver str

    name of the driver
    duration float

    duration of the charging session in hours
    email str

    email of the driver
    payment_amount float

    payment amount in PHP
    payment_method str

    payment method used
    conn_type str

    type of the connector used for the charging session

    Notes:

    • Ongoing transactions will not have payment_amount associated with them.
    • The payment_method should still reflect what the user selected upon starting a charging session
    • The duration parameter will be continuously updated upon request.
